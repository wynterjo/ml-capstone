# Student Performance Prediction Model
**Machine Learning Capstone Project | Sheridan College**

## Project Overview
This project demonstrates the application of Linear Regression to predict academic exam scores based on study hours. It was developed to showcase core AI/ML competencies, including data wrangling, predictive modeling, and statistical evaluation.

## Technical Proof of Concept
* **Model Type:** Linear Regression
* **Key Metric:** R² Score of **0.9948** (indicating 99.48% variance explanation)
* **Prediction Result:** For a target input of 10 study hours, the model predicted a score of **110.60%**.

## Tech Stack
* **Language:** Python
* **Libraries:**
  * `NumPy`: For data array manipulation.
  * `Scikit-Learn`: For model implementation and evaluation.
